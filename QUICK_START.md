# Быстрый старт

## Шаг 1: Запуск приложения

Дважды кликните на `run_gui.bat`

## Шаг 2: Добавление документов

1. Откройте вкладку **"Документы"**
2. Нажмите **"Загрузить из файла"**
3. Выберите файл `test_data.txt` (или свой файл на русском)
4. Выберите векторную БД: **Pinecone** или **Weaviate**
5. Нажмите **"Добавить документ"**
6. Дождитесь сообщения "Документ успешно добавлен"

## Шаг 3: Поиск

1. Откройте вкладку **"Поиск"**
2. Введите вопрос на русском, например:
   - "Что такое Python?"
   - "Расскажи про машинное обучение"
   - "Что такое RAG?"
3. Выберите ту же БД, куда добавляли документы
4. Установите количество результатов (например, 3)
5. Нажмите **"Поиск"**

## Результат

Вы увидите релевантные чанки из ваших документов на русском языке с оценкой релевантности.

## Если видите английский текст

Это значит в БД остались старые английские документы. Очистите индекс:

```bash
python clear_index.py
```

Затем добавьте документы заново через GUI.
