# Руководство по использованию GUI приложения

## Запуск приложения

```bash
# Активируйте виртуальное окружение
venv\Scripts\activate

# Запустите GUI
python gui_app.py
```

## Интерфейс приложения

Приложение состоит из 3 вкладок:

### 1. Вкладка "Документы"

**Добавление документов в векторную БД:**

1. Введите текст в текстовое поле ИЛИ нажмите "Загрузить из файла"
2. Выберите векторную БД (Pinecone или Weaviate)
3. Нажмите "Добавить документ"
4. Документ будет добавлен в выбранную БД и появится в списке

**Функции:**
- Загрузка текста из .txt файлов
- Добавление документов в Pinecone или Weaviate
- Просмотр списка добавленных документов
- Очистка списка документов

### 2. Вкладка "Поиск"

**Выполнение поиска:**

1. Введите ваш вопрос в текстовое поле
2. Выберите векторную БД для поиска
3. Установите количество результатов (1-10)
4. Нажмите "Поиск"
5. Результаты появятся с оценкой релевантности

**Пример запросов:**
- "Что такое машинное обучение?"
- "Как работают нейронные сети?"
- "Объясни RAG систему"

### 3. Вкладка "Сравнение"

**Сравнение векторных БД:**

1. Введите запрос для сравнения
2. Установите количество результатов
3. Нажмите "Сравнить"
4. Увидите результаты из обеих БД рядом

Это помогает понять, какая БД лучше работает для ваших данных.

## Примеры использования

### Пример 1: Создание базы знаний

1. Откройте вкладку "Документы"
2. Нажмите "Загрузить из файла" и выберите ваш .txt файл
3. Выберите "pinecone" в качестве БД
4. Нажмите "Добавить документ"
5. Повторите для всех ваших документов

### Пример 2: Поиск информации

1. Откройте вкладку "Поиск"
2. Введите: "Как установить Python?"
3. Выберите "pinecone"
4. Установите "3" результата
5. Нажмите "Поиск"
6. Изучите найденные документы

### Пример 3: Сравнение БД

1. Откройте вкладку "Сравнение"
2. Введите: "Что такое искусственный интеллект?"
3. Установите "5" результатов
4. Нажмите "Сравнить"
5. Сравните результаты Pinecone и Weaviate

## Советы

- **Добавляйте метаданные**: Документы автоматически получают ID
- **Используйте чанкинг**: Для больших текстов разбивайте на части
- **Экспериментируйте**: Попробуйте разные запросы и настройки
- **Сравнивайте БД**: Узнайте, какая лучше для ваших задач

## Горячие клавиши

- `Ctrl+O` - Загрузить файл (в разработке)
- `Ctrl+S` - Сохранить результаты (в разработке)
- `Ctrl+Q` - Выход (в разработке)

## Устранение проблем

### Приложение не запускается

```bash
# Переустановите PyQt6
pip uninstall PyQt6 -y
pip install PyQt6
```

### Ошибка "API key is required"

Проверьте файл `.env` - все ключи должны быть заполнены.

### Медленная работа

- Уменьшите количество результатов (top_k)
- Используйте более короткие документы
- Проверьте интернет соединение
